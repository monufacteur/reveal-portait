How the Template Works

🖥️

Landscape Slides

Standard Reveal.js 960 × 700 slides.
No extra markup needed.

📱

Portrait Slides

Add class="portrait" to <section> and wrap content in .portrait-content.

🖨️

Print PDF (?print-pdf)

Portrait slides are printed with the same 90° rotation as seen on screen, filling the landscape page.

⚡

Interactive Charts

Plotly.js charts work in both orientations and resize automatically.

CSS magic: In presentation mode .portrait-content is rotated 90° and scaled to fit the viewport (scale ≈ slide-height / slide-width). In print-PDF mode the scale is omitted so the rotated content fills the landscape page completely.
Portrait

Portrait Slides in Depth

Portrait slides are ideal for content that is naturally tall:

  • Tall infographics and vertical data-flows
  • Long-form documents and report excerpts
  • Vertical timelines and milestone charts
  • Portrait-oriented photography
  • Code listings that benefit from vertical space
  • Mobile-first design mockups
Presentation mode: The content box is rotated 90° clockwise and scaled to fit the viewport (transform: rotate(90deg) scale(height/width)). The scaled content is centred in the landscape slide with a small margin on all four sides.
Print / PDF mode: Append ?print-pdf to the URL and print from the browser. The 90° rotation is preserved without extra scaling so the PDF fills the landscape page completely.
Auto-wrap: If you omit .portrait-content, portrait-slides.js wraps all child nodes automatically (except <aside class="notes">).

Template Summary

Files

  • portrait-slides.css — rotation styles (presentation & print)
  • portrait-slides.js — dimension sync, auto-wrap, Plotly resize
  • index.html — this example presentation

Minimal markup

<section class="portrait">
  <div class="portrait-content">
    <!-- your portrait content -->
  </div>
</section>
PDF export: open this file with ?print-pdf appended to the URL, then use the browser's print dialog (Ctrl / ⌘ + P). Portrait slides are printed with the same 90° rotation, filling the landscape page completely.
Custom slide sizes: set width and height in Reveal.initialize() — the JS will automatically update the CSS custom properties used for the rotation and scale maths.
